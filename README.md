# celljs creator skill

用于创建 celljs 项目，示例：

```md
调用 celljs-creator 创建 celljs 项目
- celljs 版本为 3.7.1
- 后端模式为 api
- 前端框架为 react
```

可以仅创建前端若后端项目，也可是全栈项目。后端模式为 rpc/api 时，前端框架可选值为 react/vue，后端模式为 mvc 不包含前端。

参考链接：[celljs](https://github.com/cellbang/cell)

## English Usage

Used for creating celljs projects, example:

```md
Call celljs-creator to create a celljs project
- celljs version is 3.7.1
- backend mode is api
- frontend framework is react
```

You can create only frontend or backend projects, or full-stack projects. When backend mode is rpc/api, frontend framework options are react/vue, backend mode mvc does not include frontend.

Reference link: [celljs](https://github.com/cellbang/cell)
